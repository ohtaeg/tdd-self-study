# Vending Machine

음료 자판기에 탑재될 거스름돈 반환 프로그램
- 조건
    1) 최소 개수의 동전으로 잔돈을 출력한다. <br>
        예) 1000원 넣고 650원짜리 음료를 선택했다면, 잔돈은 100, 100, 100, 50원으로 반환 한다.
    2) 지폐를 잔돈으로 반환하는 경우는 없다고 가정한다. <br>
    
스스로 구현 한다. <br>
